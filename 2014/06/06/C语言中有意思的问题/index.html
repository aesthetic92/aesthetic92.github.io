%d,%f输入输出的问题

这两句输出的结果有点意外，比如第一个输出的是0.000000，而8/5本来按整数的结果是1，原因是整数和浮点数的编码方式不一样，这个整数1存储到浮点数时，编码方式完全不一样，浮点数第一位是符号位，中间8位是阶码，后面23为是尾数.

整数1按照浮点数解析表示：0 0000 0000 0000 0000 0000 0000 0000 001
表示成整数：[1 + 2^23] * 2^(-127) 很小的值

Tips：整数 / 整数 = 整数， 浮点数 / 浮点数 = 浮点数

C语言中 *号的用途

在C语言中 * 号有三个用途

  1. 乘号，用做乘法的运算，i.g: 5*6;
  2. 申明一个指针，在定义指针变量时使用，i.g: int *p;
  3. 间接运算符，取得指针所指向的内存中的值，i.g: printf("%d", *p).

Question: 指针用来存储内存地址的，为什么还要分不同类型的指针？
Answer: 因为指针变量存储的是一个内存空间的首地址（第一个字节的地址），但是这个空间占用了多少字节，用来存储什么类型的数，则是由指针的类型来标明的. 这样系统才知道应该取多少个连续内存作为一个数据.
